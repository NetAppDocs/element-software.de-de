= TestLocalConnectivity
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können das verwenden `TestLocalConnectivity` Methode zum Ping der Cluster-IP (CIP) jedes Knotens in einem aktiven Cluster.



== Parameter

Diese Methode hat keine Eingabeparameter.



== Rückgabewert

Diese Methode hat den folgenden Rückgabewert:

|===
| Name | Beschreibung | Typ 


 a| 
Details
 a| 
Individuelle Ping-Reaktionszeiten für jeden Node im lokalen, aktiven Cluster.
 a| 
JSON-Objekt

|===


== Anforderungsbeispiel

Anforderungen für diese Methode sind ähnlich wie das folgende Beispiel:

[listing]
----
{
    "method": "TestLocalConnectivity",
    "params": {},
    "id": 1
}
----
