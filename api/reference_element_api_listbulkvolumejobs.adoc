= ListBulkVolumeJobs
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können das verwenden `ListBulkVolumeJobs` Methode, um Informationen zu jedem Lese- oder Schreibvorgang des Massenvolumens im System zu erhalten.



== Parameter

Diese Methode hat keine Eingabeparameter.



== Rückgabewert

Diese Methode hat den folgenden Rückgabewert:

|===
| Name | Beschreibung | Typ 


 a| 
SperVolumes
 a| 
Ein Array von Informationen für jeden Massenvolumenjob.
 a| 
xref:reference_element_api_bulkvolumejob.adoc[BulkVolumeJob] Array erledigen

|===


== Anforderungsbeispiel

Anforderungen für diese Methode sind ähnlich wie das folgende Beispiel:

[listing]
----
{
   "method": "ListBulkVolumeJobs",
   "params": {
      },
   "id" : 1
}
----
